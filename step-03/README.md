# ステップ3: 基本的な評価器

## 目的

構築した構文木を評価（実行）して、結果の値を得ます。
このステップでは、基本的な数値演算を実装します。

## 学ぶこと

- 構文木の評価（インタープリター）の基本
- 環境（環境変数の管理）
- 基本的な関数適用
- 数値演算の処理

## 実装内容

構文木 `'(+ 1 2)` を評価して、結果 `3` を得ます。

## 評価の基本

1. **数値**: そのまま返す
2. **シンボル**: 環境から値を取得
3. **リスト**: 関数適用として評価

## 対応する演算

- `+`: 加算
- `-`: 減算
- `*`: 乗算
- `/`: 除算

## 実装のポイント

1. 再帰的に式を評価する
2. 関数適用時、最初の要素（関数名）を見て処理を分岐
3. 引数を先に評価してから演算を実行
