# ステップ7: 再帰とループ構文

## 目的

再帰的な関数呼び出しをサポートし、`let` や `letrec` などの構文を実装します。
これにより、より複雑なプログラムが書けるようになります。

## 学ぶこと

- 再帰関数の実装
- `let`: 局所変数の定義
- `letrec`: 相互再帰を含む局所定義
- ループ構文（オプション）

## 実装内容

以下のようなコードが評価できるようになります：

```scheme
;; 再帰的な階乗関数
(define factorial
  (lambda (n)
    (if (= n 0)
        1
        (* n (factorial (- n 1))))))

(factorial 5)  ; => 120

;; let を使った局所変数
(let ((x 10) (y 20))
  (+ x y))  ; => 30
```

## 実装のポイント

1. 再帰関数は環境を正しく管理することで動作
2. `let` は新しい環境を作成して式を評価
3. `letrec` は相互再帰を可能にするため、環境を拡張してから値を評価
