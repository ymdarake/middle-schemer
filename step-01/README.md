# ステップ1: レキサー（字句解析）

## 目的

文字列のプログラムコードを、意味を持つ最小単位（トークン）に分割する処理を実装します。
これが言語処理系の最初のステップです。

## 学ぶこと

- 文字列をトークンに分割する方法
- Schemeの文字列操作とパターンマッチング
- 基本的な再帰的処理

## 実装内容

入力文字列 `"(+ 1 2)"` を以下のようなトークンリストに分割します：

```scheme
'("(" "+" "1" "2" ")")
```

## トークンの種類

- **開き括弧**: `(`
- **閉じ括弧**: `)`
- **識別子**: `+`, `-`, `*`, `/`, `define`, `lambda` など
- **数値**: `1`, `2`, `3.14` など
- **文字列リテラル**: `"hello"` など

## 実装のポイント

1. 空白文字をスキップする
2. 括弧を個別のトークンとして認識する
3. 識別子（英数字と記号の組み合わせ）を抽出する
4. 数値を正しく認識する

### 応用問題

1. **識別子の拡張**: アンダースコア（`_`）も識別子に含めるように拡張してください。

2. **負の数値**: 負の数値（例: `-42`）を認識できるように拡張してください。

3. **文字列リテラル**: ダブルクォートで囲まれた文字列リテラル（例: `"hello"`）を認識できるように拡張してください。

4. **コメント**: `;` から行末までのコメントをスキップできるように拡張してください。
