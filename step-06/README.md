# ステップ6: 条件分岐とブール値

## 目的

`if` 文とブール値（`#t`, `#f`）を実装して、条件に基づいた処理を可能にします。

## 学ぶこと

- 条件分岐の実装
- 真偽値の扱い
- 比較演算（`=`, `<`, `>`, `<=`, `>=` など）
- 論理演算（`and`, `or`, `not`）

## 実装内容

以下のようなコードが評価できるようになります：

```scheme
(if (> 5 3) 10 20)  ; => 10
(if (< 5 3) 10 20)  ; => 20
(if (and (> 5 3) (< 2 4)) 100 200)  ; => 100
```

## 真偽値の扱い

- `#t`: 真（true）
- `#f`: 偽（false）
- Schemeでは、`#f` と空リスト `()` のみが偽、それ以外はすべて真

## 実装のポイント

1. `if` 式の評価: 条件を評価し、真なら第2引数、偽なら第3引数を評価
2. 比較演算: 数値同士を比較する関数を実装
3. 論理演算: 短絡評価（short-circuit evaluation）を実装

## 実行方法

```bash
# Racketの場合
racket step-06/conditionals.scm
```

## 練習問題

1. **三項演算子**: `if` を使って三項演算子のような関数を作成してください。

2. **条件式の連鎖**: 複数の条件を連鎖させる関数を作成してください。

3. **パターンマッチング**: 値のパターンに応じて処理を分岐する関数を作成してください。
